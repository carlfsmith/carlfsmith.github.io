Side-story Editor

Android App

Developed for Android 6 (Marshmallow), this application is a text editer developed in Java using Android Studio which allows users to develop various branching storylines. Each text component is grouped in a 'scene' which can lead to one or more follow-up scenes which can contain their own story branches. The editor also handles scene-tree restructuring when scenes are deleted or moved. Currently in development is adding cloud scene sharing functionality.

Evolutionary Path Traversal

A.I. Programming

Program created using Python which has competing members of a population generate relative minimum cost paths over generations. Each 'individual' is a chromosome composed of genes which determine a number of steps to take and an associated angle. Each generation one member dies off and another is created using genes from two other member of the population. Pseudo randomness is also added to the system to add/drop genes or mutate them. A graphing API (Xlsxwriter) was also added to chart minimum costs in the population over time.

Tile-map Editor

2-D Level Editor

Using Python and Pygame API, created a program which allows the user to paint 2D game tile maps from imported tile-sets of varying bit sizes. Imported tile-sets of any size would be partially displayed in a work-space coupled with an editor window of user defined size. UI elements, such as sub-windows and scroll bars were programmed using primative shapes. A save feature was included to output the user constructed tile-map.
